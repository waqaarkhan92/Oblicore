## Registry Purpose

This registry tracks all authoritative source documents referenced by ingestion prompts using the `[XX-###]` citation format. Each source has been verified and assigned a unique authority ID.

## Registry Maintenance

### Status Definitions

| Status | Definition |
|--------|------------|
| Active | Source document is current and valid |
| Superseded | Replaced by newer version (reference retained for historical prompts) |
| Withdrawn | No longer published by authority |
| Pending | Awaiting confirmation from authority |

### Update Protocol

1. Review all sources quarterly for updates
2. Update version year in prompt when source refreshed
3. Mark superseded sources but retain for audit trail
4. Add new sources as regulatory landscape evolves
